# マルチワールドHTTPサーバ

ここでは、multi world gcを使って、軽量プロセスによるwebサーバを作成します。
目標は、プログラムはDLLとして読み込み、軽量メモリ空間で実行され、処理が終わると解放する事です。
プログラムが古ければコンパイルし直されます。
非同期IOが理想ですが、そこまではしません。


	make http

として、


	http://localhost:8088/string

にアクセスすると、string.cがコンパイルされて実行されます。
